# Use Case: Produce Report of number of people who speak Chinese, English, Hindi, Spanish and Arabic from greatest number to smallest including the percentage of the world population

## Characteristic Information

### Goal in Context

As a *user*, I want *to produce a report for the number of people who speak Chinese, English, Hindi, Spanish and Arabic from greatest number to smallest including the percentage of the world population* so that *I can gain insights of global language distributions*.

### Scope

Global language data.
### Level

Primary task.

### Preconditions

The user has access to global language data.
The database has up-to-date data of people categorized by language.

### Success End Condition

A report is produced for the number of people who speak Chinese, English, Hindi, Spanish, and Arabic from greatest number to smallest, including the percentage of the world population. 

### Failed End Condition

No report is generated.

### Primary Actor

User.

### Trigger

The user requests the system for the number of people who speak different languages in the world.

## Main Success Scenario

1. The user requests the system to produce the report for the number of people who speak Chinese, English, Hindi, Spanish and Arabic from greatest number to smallest including the percentage of the world population.
2. System retrieves global language data.
3. System extracts population data categorized by language.
4. System calculates the number of people who speak Chinese, English, Hindi, Spanish, and Arabic.
5. System organizes those population data from greatest to smallest based on the number of speakers.
6. System calculates the percentage of the world population for each population data.
7. System generates a report for the number of people who speak Chinese, English, Hindi, Spanish, and Arabic, along with the percentage of the world population.
8. User receives the report.
